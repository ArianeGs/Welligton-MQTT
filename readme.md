## Installation
### Enviroment configuration

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install pika.

```python
#install the venv into your project folder
py -m venv venv 
or 
python -m venv venv

# install the pika lib
pip3 install pika

```
